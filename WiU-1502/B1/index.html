project name: WorldSurf

project description

Welcome to the Worldsurf.com project page. Here you will find the most current links to the usability tests being conducted along with the data they produce. Usability Testing will be conducted in order to begin the project with a solid foundation. This testing allows us to get inside the minds of the target audience and we gain an understanding of what works and how to use it to engage and interact with the user.

My goal with the WorldSurf project is to guide the user to their intended goals on WorldSurf with an intuitive, functional, and effective user interface to achieve the best user experience possible. I will be utilizing Usability Hub's NavFlow tests and analyze the data which provides an invaluable insight into the minds, behaviors, and intentions of the project's target audience.
